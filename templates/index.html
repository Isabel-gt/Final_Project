Introduction

Our project focus on using machine learning to pre diagnose patients who might be at risk of diabetes since the disease can lead to serious complications such as heart disease, vision loss, lower-limb amputation and kidney disease. The predictions can assist clinicians formulate a In this project, we intend to use supervised machine learning (classification)to predict diabetes or no diabetes.

Method

We selected a dataset from the Kaggle webpage and decided to use Flask and HTML to build a web page to channel and test the machine learning model. We chose a template from Start Bootstrap and worked on editing and customizing it. We built 2 databases using sqlite, one with all the data and one with only 10 features. We used python in vs code and jupyter notebook to build, and train the models and for vizualization charts as well. Finally, we built flask routes to the html, the database and the machine learning model in order to facilitate testing and demostration.

Dataset Choice

We used the Diabetes Health Indicators Dataset from 2015 provided by Kaggle. The dataset was obtained by surveying hundreds of thousands of Americans for diabetes or prediabetes predictors with The Behavioral Risk Factor Surveillance System (BRFSS). We decided to explore 10 of the 22 variables for our prediction; education, age, high BP, gender, BMI, cholesterol check, no doctor because of cost, stroke, smoker, physical activity.

Model Evaluation

The project evaluated a deep learning approach using TensorFlow's deep neural network (DNN) module.The model consists of an input layer, a dense layer with sigmoid activation function, and an output layer. We also evaluated using the logistic regression model as a supervised machine learning model that classify the predictions into binary outcomes. We performed a feature importance on the original dataset in order to rank the features and how they affect the model.

Model Selection

This section evaluate which machine learning model provide the best accuracy. Ideal picture screenshot or chart below.

  • Step 1

    Deep Learning

    TensorFlow sigmoid function model was instantiated. The data was slip 75/25 into training and testing set. We use 3 hiddden layers. and returned an accuracy score of 86%.

  • Step 2

    Logistic Regression

    The logistic regression model was chosen as a supervised machine learning model.The data was split 75/25 into training and testing data. The model was instantiated, the training data was fit into the model and a prediction was run on the testing data. The model returned an accuracy score of 85.9287%.

  • Step 3

    Sqlite Databases

    We build an original database with 22 features using CREATE TABLE and from there we select 10 features to include in the model and built a second table. We used connection and cursor methods in flask to link the table.

  • Step 4

    Flask Routes & HTML Edits

    Finaly, we built flask routes to the html homepage, to the database page and to model testing form on the webpage. We also edited and customized the page to out project.
